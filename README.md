# minesweeper



지뢰찾기게임을 javascript로 구현
jquery 사용

사용방법
	마우스 왼쪽 클릭 		          ==> 	셀 오픈(확인)
	
	마우스 오른쪽 클릭 	         	 ==>	지뢰 표시(깃발)
	
	마우스 스크롤 클릭		         ==>	모를 때(물음표)
	
	마우스 양쪽 클릭		          ==>	주변 8개 셀 오픈(확인)
	
	new Game 버튼		              ==>	현재 난이도로 새로운 게임
	
	좌측 상단 easy, normal, hard      	    ==>	선택한 난이도로 새로운 게임



코드 설명
	지뢰찾기의 칸 하나 ==> cell
	cell은 map 자료구조를 활용하였다.
